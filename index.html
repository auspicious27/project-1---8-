Modules 1 to 8 Integrated Project

CampusConnect Course Enquiry System

A complete teaching project that covers Web Fundamentals, HTML5, CSS3, JavaScript, Advanced JavaScript, Node.js, and Express REST APIs in one practical flow.

What Are We Building?

We will build a project called CampusConnect Course Enquiry System. It has a responsive frontend website where students can view courses and submit an enquiry form. JavaScript validates the form, stores data in browser localStorage, and also sends the data to a backend API. The backend is built using Node.js and Express.js.

Is project ko aap learners ko step-by-step padha sakte ho. Har module ka topic project ke kisi practical part se connect hoga. Isse learners ko theory bhi samajh aayegi aur unka real project bhi complete ho jayega.

Why Are We Building This Project?

Hum ye project isliye build kar rahe hain kyunki learners ko HTML, CSS, JavaScript, Node.js, and Express alag-alag topics ki tarah nahi, balki ek real application ke parts ki tarah samajhna chahiye. Real world mein koi bhi website sirf HTML se complete nahi hoti. HTML structure deta hai, CSS design deta hai, JavaScript behavior deta hai, backend request handle karta hai, and API frontend-backend ko connect karti hai.

CampusConnect project ek course enquiry system hai. Training institute, coaching center, school admission team, online course platform, ya workshop registration website mein aisa system real use hota hai. Student form fill karta hai, frontend data collect karta hai, validation wrong input ko stop karti hai, backend data receive karta hai, and admin later enquiry data dekh sakta hai.

Complete Project Flow in Simple Story

Socho Rahul naam ka student website open karta hai. Browser sabse pehle index.html load karta hai. HTML page ka structure banata hai: navbar, course cards, enquiry form, dashboard. Uske baad browser styles.css load karta hai. CSS page ko professional design deta hai: colors, spacing, cards, buttons, responsive layout. Phir browser script.js load karta hai. JavaScript page ko interactive banata hai: form ka data read karta hai, validation karta hai, localStorage mein save karta hai, dashboard update karta hai, and backend API ko request bhejta hai.

Backend side par server.js run hota hai. Jab frontend POST /api/enquiries request bhejta hai, Express route request receive karta hai. express.json() middleware JSON body read karta hai. Backend name, email, course, message validate karta hai. Agar data valid hai toh enquiry array mein save hoti hai and response return hota hai. Frontend response read karke student ko success message show karta hai.

function validateForm(enquiry) {
  if (enquiry.name.length < 3) {
    return "Name must be at least 3 characters.";
  }

  if (!enquiry.email.includes("@")) {
    return "Please enter a valid email address.";
  }

  if (enquiry.course === "") {
    return "Please select a course.";
  }

  if (enquiry.message.length < 10) {
    return "Message must be at least 10 characters.";
  }

  return "";
}
Then Add Submit Event In: frontend/script.js 
enquiryForm.addEventListener("submit", async function (event) {

  });

app.post("/api/enquiries", function (request, response) {
  const name = request.body.name;
  const email = request.body.email;
  const course = request.body.course;
  const message = request.body.message;

  const newEnquiry = {
    id: enquiries.length + 1,
    name: name.trim(),
    email: email.trim(),
    course: course,
    message: message.trim()
  };

  enquiries.push(newEnquiry);

  response.status(201).json({
    success: true,
    message: "Enquiry saved successfully on backend.",
    data: newEnquiry
  });
});

app.listen(PORT, function () {
  console.log("CampusConnect API running on http://localhost:" + PORT);
});

Common Errors

  • CSS file not loading: check href="styles.css".
  • JavaScript not working: check script.js path and browser console.
  • Form refreshes page: add event.preventDefault().
  • Backend request fails: check if node server.js is running.
  • Express body undefined: add app.use(express.json()).
  • CORS issue: backend must set CORS headers or use CORS middleware.
  • Port already in use: stop old server using Ctrl + C.
