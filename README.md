# Full Stack Web Project - Module 1 to 8 Notes

This folder contains a detailed classroom-style project for teaching Modules 1 to 8 through one practical full stack flow.

Project name: CampusConnect Course Enquiry System

The notes explain why the project is built, real-world use cases, frontend/backend flow, step-by-step execution, expected outputs, common errors, and detailed code explanation in Simple Hinglish.

## Files

- `index.html` - Complete classroom notes in Simple Hinglish.
- `campus-connect-project/frontend/` - HTML, CSS and JavaScript frontend.
- `campus-connect-project/backend/` - Node.js and Express REST API backend.

## What This Project Teaches

- Web and internet request-response flow
- HTML5 semantic structure
- CSS3 responsive styling
- JavaScript DOM, events, validation and localStorage
- Node.js backend setup
- Express REST API routes
- Frontend to backend connection using fetch

## Run Frontend

Open:

`campus-connect-project/frontend/index.html`

## Run Backend

```bash
cd campus-connect-project/backend
npm install
node server.js
```

Backend URL:

`http://localhost:3000`

## Classroom Demo Flow

1. Open frontend page and explain HTML/CSS.
2. Submit wrong form data and show validation.
3. Submit valid form data and show local dashboard update.
4. Start backend server.
5. Submit form again and show backend success response.
6. Open `/api/enquiries` and show JSON API output.
